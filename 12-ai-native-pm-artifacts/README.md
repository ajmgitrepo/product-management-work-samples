# AI-Native Product Management Portfolio Artifacts

This package contains three portfolio-ready artifacts positioning an AI-native Product Manager focused on cloud platforms, developer experience, infrastructure automation, and agentic workflows.

## Artifacts

1. **Agentic Cloud Onboarding PRD**  
   A product requirements document for an AI-assisted platform that helps application teams onboard containerized services across Azure and GCP.

2. **AI Product Evaluation Framework**  
   A practical rubric for evaluating AI features across business value, user trust, operational quality, cost, governance, and product adoption.

3. **AI-Native PM Operating System**  
   A repeatable workflow for how an AI-native PM uses AI across discovery, strategy, execution, analytics, stakeholder communication, and portfolio development.

## Positioning Theme

> AI-native Product Leader focused on cloud infrastructure, developer platforms, automation, and agentic workflows.

These artifacts are designed for interview prep, GitHub portfolio use, and product leadership storytelling.
